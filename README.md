# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

## Environment Variables

To use the AI features in this project, you will need to add the following environment variables to a `.env` file at the root of your project:

- `GEMINI_API_KEY`: Your API key for Google Gemini. You can get a key from [Google AI Studio](https://aistudio.google.com/app/apikey).
